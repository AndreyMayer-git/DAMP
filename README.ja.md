## 概要

DAMP は、短時間提示された感情画像間の強制選択における*ダイナミクス*を記録・分析し
ます。最終的な選択だけでなく、決定がどのように展開するか——運動競合（マウス軌跡）と
選択の安定性——をプロセスレベルの指標として捉えます。研究・探索目的のみで、臨床的・
診断的な解釈は想定していません。

## 実験デザイン

**刺激。** OASIS データベース（Kurdi et al., 2017）の画像を感情価
（**ポジティブ／ネガティブ／ニュートラル**）で分類し、前処理します：背景除去、
グレースケール化、ガウシアンぼかし、単一のフラットカラーでの描画。**色は感情価とは
独立に割り当てられ**（カテゴリ間で交差）、色相や輝度が感情カテゴリの代理にならない
ようにします。処理後のセットは、刺激作成者とは別の評価者が（理想的には感情価に対して
盲検で）手作業で精選します（対象が判別可能であること、キャンバスの約 40% 以上を
占めること）。

**試行の流れ。** (1) 警告音 → (2) 注視点（2.0–3.5 秒）→ (3) 2 つの刺激を逐次提示
（タイミングは下記）→ (4) 空白画面での反応ウィンドウ 3.0 秒 → (5) 確認 + 試行間
間隔（1.0–2.5 秒）。マスキングは使用しません。

**タイミング**（予期を防ぐため試行ごとにジッター）：

| パラメータ | 範囲 |
|---|---|
| 刺激呈示時間（各） | 18–30 ms（165 Hz で 3–5 フレーム） |
| 刺激間間隔（ISI） | 80–150 ms |

**試行タイプと構成：**

| タイプ | 数 | |
|---|---|---|
| ポジティブ–ニュートラル | 17 | ニュートラル含む |
| ネガティブ–ニュートラル | 17 | ニュートラル含む |
| ポジティブ–ネガティブ | 17 | 両方が感情価 |
| SWAP（鏡像での再提示） | 10 | 安定性チェック |
| **合計** | **61** | |

> **ラベルについて。**「両方が感情価」対「ニュートラル含む」は、DCI コントラストを*刺激の構成*によって定義します。両方が感情価をもつ条件が実際により大きな軌跡逸脱（DCI > 0）を生むかどうかは、下記の未解決の経験的問題であり、前提ではありません（プレプリント §2.2 / §4.2）。

見逃し試行（3.0 秒以内に反応なし）はセッション末尾で再提示され、主要分析からは除外
されます。

## キャリブレーション

ランダムに左右へ現れる四角をクリックする 7 試行。その RT の中央値を個人のベースライン
（`base_rt`）とし、RT 正規化に用います。

## 記録

- 各試行を通じた **100 Hz**（10 ms ごと）のマウス軌跡
- 反応時間（反応ウィンドウ開始からクリックまで）
- 刺激ごとの色カテゴリ（記述的バイアス分析）

## 行動指標

**主要指標：**

| 指標 | 式 | 解釈 |
|---|---|---|
| **DCI** — 決定葛藤指標 | `(AUC_high − AUC_low) / AUC_low` | 両方が感情価 vs ニュートラル含む条件での相対的な軌跡逸脱。DCI > 0 が成り立つかは**未解決の経験的問題**であり、ここでは確立されていません（プレプリント §2.2 / §4.2 参照）。集団推論には、この比よりも差分コントラスト `AUC_high − AUC_low` の方が頑健です。 |
| **RSI** — 表象安定性指標 | `N_same / N_swap` | 空間反転にもかかわらず元の選択を維持した SWAP 試行の割合。偶然水準を上回る ⇒ 選好の安定性（信頼区間で評価）。 |

AUC = 始点と終点を結ぶ理想直線からの垂直方向の逸脱の積分（台形則）。DCI では `AUC_high` は両方が感情価をもつ条件（ポジティブ–ネガティブ）、`AUC_low` はニュートラルを含むペアです。「高/低」は刺激の構成を指すラベルであり、葛藤の大きさを仮定するものではありません。

**探索的指標：**

| 指標 | 定義 | 注記 |
|---|---|---|
| **FLIP** | 軌跡が中央の垂直軸（x = 0）を横切る回数を試行平均したもの | 記録はされるが**主要分析の枠組みからは除外**——理論的特異性が低く、運動ノイズに敏感（プレプリント §4.1）。 |

## 計算モデルとシミュレーション

プレプリントには**人を対象とするデータはありません**。すべてのモデル図は第 2.2 節の
生成モデルからのシミュレーションです。2 つの競合する反証可能なモデルが、予測される
**DCI の符号**を挟み込みます：

- **モデル 1 — 最小ドリフト拡散**（`dampsimulation.py`）：**DCI ≤ 0** を予測
  （最大の感情価差をもつポジティブ–ネガティブが最速で解決 → 最も直線的な経路）。
  ノイズパラメータ σ に対して頑健。
- **モデル 2 — leaky competing accumulator**（`damp_lca_model.py`; Usher &
  McClelland, 2001）：顕著性捕捉の重み γ > 0 で **DCI > 0** を予測。

したがって DCI の符号は、シミュレーションの結果ではなく、将来のデータに対する経験的な
問いです。

| スクリプト | 目的 |
|---|---|
| `main.py` | 実験の実行（PsychoPy）：同意、キャリブレーション、61 試行 |
| `analyze_data.py` | `data/` 内の各フォルダについて指標 + 図を計算 |
| `dampsimulation.py` | モデル 1（最小 DDM） |
| `damp_lca_model.py` | モデル 2（LCA + 顕著性重み付き競合）；γ スイープ |
| `dampmakefigure.py` | 図 2–5 |
| `damp_model_comparison_figure.py` | 図 6（DDM vs LCA + 創発的な高葛藤軌跡） |
| `damp_recovery.py` | 図 7（モデル 1 のパラメータ回復） |
| `damp_power_analysis.py` | ブートストラップ検出力分析 |
| `damp_extra_checks.py` | DCI 符号の σ 頑健性、AUC の二峰性、効果量ごとの検出力 |
| `Database_2026/script.py` | 刺激生成（背景除去、グレースケール、ぼかし、感情価 ⊥ のフラットカラー） |
| `Color_images.py` | 各刺激に支配色をラベル付け |

図は `figures/`（図 2–7）にあり、**すべてシミュレーション**です。


## セットアップ

**Python 3.13。** リフレッシュレート **165 Hz 以上**のモニタ（フレーム単位のタイミング）、
標準的なマウス、固定された視距離（約 1 m）。

```bash
# 実験・分析・シミュレーション:
pip install psychopy numpy pandas matplotlib seaborn sounddevice

```

## 使い方

1. OASIS の元画像を感情価ごとに `Database_2026/images/` に置きます。
2. `Database_2026/script.py` を実行して標準化します（背景除去、グレースケール、
   ぼかし、フラットカラー）。出力は `Database_2026/processed_stimuli/` に生成され
   ます。
3. `processed_stimuli/` を手作業で精選します：対象が判別可能で、キャンバスの約 40%
   以上を占める画像のみ残し、それ以外は削除します。この作業は刺激作成者とは別の
   評価者が、理想的には感情価に対して盲検で行います。残した画像を
   `Database_2026/final_standardized_stimuli/` に移し、`Positive/`, `Negative/`,
   `Neutral/` のサブフォルダ（各 50 枚以上）に分けます。
4. `main.py` を実行——参加者 ID、年齢、性別を入力し、同意とキャリブレーションを
   経て、本課題（約 40 分）へ。
5. `analyze_data.py` を実行し、`data/` 内の全セッションについて指標と図を生成します。

**パイロットモード**（`use_fixed_stimuli = True`）：`pilot_template.json` により全
参加者で刺激とタイミングを同一化（無ければ自動生成）。
**ランダムモード**（`= False`）：セッションごとに完全ランダム化。
